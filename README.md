# FINTECH
install

> npm i

to run in develop mode, you have to use two console, one for build, and the other to run

> npm run build
> npm run dev

to test

> npm test -- -w

to run 

> npm run start

env content

FINTECH_SERVER_HTTP_PORT = 3001
OPEN_WEATHER_URL = https://api.openweathermap.org/data/2.5
IP_API_URL = http://ip-api.com/json
